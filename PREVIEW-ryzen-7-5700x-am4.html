📚 ¿Cores vs Threads? Entender la diferencia
¿Qué es un CORE (Núcleo)?
Un core es una unidad de procesamiento física e independiente. Con 8 cores, tu CPU puede procesar 8 operaciones completamente diferentes simultáneamente. Cada core tiene su propia lógica de procesamiento.
¿Qué es un THREAD (Hilo)?
Un thread es una línea de instrucciones que el CPU puede ejecutar. Con SMT (Simultaneous Multi-Threading), cada core puede ejecutar 2 threads en paralelo, simulando ser 2 cores lógicos.
